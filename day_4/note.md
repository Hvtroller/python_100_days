## Add element to list

```python
# Create a list
my_list = [1, 2, 3]

# Add an element to the list
my_list.append(4)
# Add multiple elements to the list using +
my_list = my_list + [4, 5]
# Add multiple elements to the list
my_list.extend([4, 5])
print(my_list)
```
